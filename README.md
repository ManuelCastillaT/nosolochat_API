# README

Intructions for 

1. Clone repository

* git clone https://github.com/ManuelCastillaT/nosolochat_API.git

2. Install gems on api

* bundle install

3. Install redis-server

* sudo apt install redis-server

4. Run mongodb server

* service mondodb start

5. Run redis server

* redis-server

6. Run frontend

* yarn dev

7. Run backend

* rails server

To run tests

* rspec
